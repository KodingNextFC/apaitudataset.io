1. Apa Itu Dataset?

Kumpulan Data yang Saling Berkaitan

Dataset adalah kumpulan data yang saling berkaitan dan disusun dalam bentuk baris dan kolom. Dataset berisi informasi yang digunakan oleh model machine learning untuk mempelajari pola dan membuat prediksi.

2. Apa Itu File CSV?

Comma-Separated Values (CSV)

File CSV menyimpan data dalam bentuk teks biasa. Setiap baris ditulis pada baris baru, dan setiap nilai dipisahkan dengan tanda koma. CSV merupakan salah satu format yang paling umum digunakan untuk menyimpan dataset.

Name,Age,StudyHours,Passed Alice,18,5,Yes Bob,19,2,No Charlie,18,4,Yes
Nama Usia Jam Belajar Lulus
Alice 18 5 Ya
Bob 19 2 Tidak
Charlie 18 4 Ya

✅ Setiap tanda koma menjadi pemisah kolom, dan setiap baris baru menjadi data baru.

Sederhana & ringan
Mudah dibuat dan diedit
Kompatibel dengan Excel & Google Sheets
Didukung oleh hampir semua bahasa pemrograman
3. Apa Itu DataFrame?

Struktur Data Berbentuk Tabel di Python

Ketika sebuah dataset yang tersimpan dalam file CSV dimuat ke dalam Python, data tersebut biasanya diubah menjadi DataFrame. DataFrame adalah struktur data berbentuk tabel yang memudahkan kita melihat, memfilter, dan menyiapkan data sebelum melatih model machine learning. Bayangkan seperti spreadsheet yang berada di dalam program Python.
